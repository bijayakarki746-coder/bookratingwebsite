# 📚 Book Rating Website

## Overview

Book Rating Website is a web application that helps readers keep track of the books they have read and remember their thoughts, ratings, and important details over time. Readers often forget how much they enjoyed a book or specific information about it after months or years. This project provides a simple way to store and review those details.

## Features

### User Features

* User registration and login system.
* Users must log in to view book details.
* Browse and view available books.
* View ratings and information about books.
* Secure access to book-related content.

### Admin Features

* Add new books to the collection.
* Edit existing book information.
* Delete books from the collection.
* View all ratings submitted for each book.
* Manage the entire book catalog.

## Technologies Used

* **HTML** – Structure of the website.
* **CSS** – Styling and responsive design.
* **JavaScript** – Application functionality and interactivity.
* **Local Storage** – Stores user, book, and rating data in the browser.

## Project Motivation

Many readers forget important details about books they have read, such as plot points, personal opinions, and ratings. This website was created to help users maintain a record of their reading experience so they can easily revisit and remember books they have enjoyed.

## How It Works

1. Users log in to access book information.
2. Admins can add, edit, and delete books.
3. Users can browse books and view details.
4. Ratings are stored and displayed for each book.
5. All data is saved using the browser's Local Storage.
